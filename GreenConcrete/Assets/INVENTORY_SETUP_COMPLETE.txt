ADVANCED USAGE

Listening to Inventory Changes:
```csharp
void Start()
{
    PlayerInventory.Instance.onInventoryChanged += OnInventoryChanged;
}

void OnInventoryChanged()
{
    Debug.Log("Inventory changed!");
    Debug.Log("Total weight: " + PlayerInventory.Instance.GetTotalWeight());

```

Checking if Item Fits:
```csharp
InventoryItem newItem = new InventoryItem(weaponDef);
Vector2Int? position = PlayerInventory.Instance.mainGrid.FindSpaceForItem(newItem);

if (position.HasValue)
{
    PlayerInventory.Instance.mainGrid.PlaceItem(newItem, position.Value.x, position.Value.y);
}
else
{
    Debug.Log("No space!");
}
```

Equipping Items:
```csharp

PlayerInventory.Instance.EquipArmor(armorItem);

```
